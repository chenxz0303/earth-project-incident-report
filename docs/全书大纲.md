# 全书大纲

> 当前为结构级大纲，不含正式章节正文。卷名、章数和具体案件可在正式创作前调整。

## 总体主线

外星歼星舰试射击穿地球底层参数，现实开始异常。仿真工程师陈默意外获得有限的现实参数修改权限，并被带上歼星舰协助调查。银河监管局即将审查事故；如果地球被认定为不可修复的损坏样本，就会被删除。陈默必须查清事故真相、修复异常，并写出一份既不撒谎又能保住地球的事故报告。

## 六卷建议结构

### 第一卷：现实参数异常（约 24—27 章）

建立陈默的工作与性格，连续出现可被仿真知识识别的现实异常。陈默获得受限修改权限，与事故联络官芙娅接触，最终被迫登上事故舰。卷末确认地球不是自然宇宙中的普通行星，而是受监管的社会仿真项目。

### 第二卷：歼星舰事故组（约 24—27 章）

陈默进入外星事故调查组，学习宇宙项目制度，开始处理结构、热流体和多物理场异常。各部门围绕责任边界互相甩锅。陈默发现试射记录与实际损伤不一致，事故可能不是单纯操作失误。

### 第三卷：地球补丁上线（约 24—28 章）

调查组回到地球实施修复，多个补丁产生连锁副作用。陈默利用工程方法把超自然异常转化为可验证问题。芙娅与陈默建立初步信任。卷末发现有人利用事故掩盖对地球样本的非法修改。

### 第四卷：银河监管预审（约 23—27 章）

监管局提前介入，要求冻结修改权限、提交证据链和整改报告。陈默在事实、措辞和地球存亡之间周旋。内部证人和关键日志相继失效，调查转为追查权限来源与利益链。

### 第五卷：被篡改的试射报告（约 24—28 章）

调查深入到歼星舰设计、试射审批和地球项目历史。结构失效与流体热异常两条证据链汇合，证明事故经过被系统性改写。陈默需要在不越权的前提下复现实验，锁定真正责任方。

### 第六卷：最终事故调查报告（约 25—30 章）

监管听证、现实修复与真相公开同步推进。陈默完成一份不撒谎但重新定义“损坏样本”的报告，使地球从待删除对象转为具有不可替代价值的在运行项目。主体结局圆满：地球获保留，陈默取得项目管理员权限；结尾揭示更高层项目仍在观察，为续作留下悬念。

## 结局约束

- 陈默保住地球并获得项目管理员权限。
- 事故真相得到足够解释，主要责任链闭合。
- 陈默与芙娅的关系有明确进展但不作喧宾夺主的大团圆处理。
- 留下高一层级项目或管理员体系的续作悬念。
