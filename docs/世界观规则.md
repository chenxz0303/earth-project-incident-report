# 世界观规则

## 核心层级

1. 地球是编号 E-731 的长期社会仿真项目，内部时间、体验、伤害和选择对居民真实有效。
2. 地球所在的参数层隶属于银河项目体系；银河文明可以维护和监管项目，但不能完全预测样本内个体选择。
3. 衡光号的因果穿透武器可以跨越普通空间作用于目标底层规则，因此试射才可能击穿地球参数层。
4. 银河监管局可以对项目作出保留、隔离、回滚或删除决定，但必须满足证据、比例和风险程序。
5. 全书结尾只暗示银河监管文明上方仍有“上层观测域”，不在本书解释其完整结构。

## 地球项目设定

- 正式编号：E-731。
- 历史别名：K-217-E，源于旧档案迁移，事故前本应永久作废。
- 原始目的：研究资源受限条件下文明如何产生合作、纠错与技术进步。
- 隐藏价值：地球产生了创建者模型无法预设的新知识，因此不能用旧备份完整替代。
- 旧维护机制：当外部维护链全部失效，系统可选择样本内居民作为最低权限应急维护人。
- 保密原则：正式剧情中大多数地球居民不知道仿真真相，避免故事无限扩张成全球政治线。

## 陈默权限规则

- 初始每日额度为 12 个因果预算点，随风险等级与监管授权临时变化。
- 日额度以事故发生时刻 T0 为刷新基准，未使用额度不结转；持续跨日的非原生补丁必须续期，并重新消耗额度。
- 只能修改已获权限、能用工程语言定义且能设计验证方法的参数。
- 不得直接指定结果，例如不能写“桥梁不倒”，只能调整可解释的阻尼、接触或边界参数。
- 作用范围越大、持续时间越长、耦合系统越多，消耗越高。
- 修改会留下不可删除的审计记录，包括发起人、依据、范围、预期结果和实际后果。
- 紧急状态允许先执行后补报告，但不免除责任。
- 修改无法复活已死亡个体、重写人格或无损回滚复杂历史。
- 结局管理员权限仍保留额度、双人复核、监管抽查和自动停止条件。
- 最终 72 小时修复采用监管封存的 36 点专项预算，等同预支三个地球日额度并按阶段释放；预算耗尽后不会因跨日再次刷新。

## 修改副作用规则

- 参数不是独立旋钮；结构、流体、热、概率和社会行为可能通过隐藏接口耦合。
- 局部修复可能把误差转移到邻近区域或其他物理场。
- 系统遵守守恒：长期被抽走或压缩的能量、数据负载和因果偏差必须重新分配。
- 越接近底层根裂纹，修改对现实的影响越大，也越难回滚。
- 地球居民的集体行为不能被当作单一确定参数，只能作为有统计范围的内生响应。

## 银河监管规则

- 监管审查采用“事实—推断—责任意见—处置建议”四层结构。
- 证据必须说明来源、版本、校验、获取权限和是否可重复。
- 项目是否保留取决于剩余风险、替代可能、独特价值、受害者权利和整改可行性。
- 监管官不得同时调查与本人利益相关的机构，发现冲突必须申请回避。
- 删除属于不可逆处置，若项目内存在事故证人或不可替代数据，应优先隔离和保全。
- 样本可以通过证明自治、纠错和自审计能力申请当事方资格。
- 样本居民无法公开授权时，监管局必须指定独立的样本权益监护人；监护人负责居民整体生命、连续体验和证据权利，不受项目所有者指挥。
- 陈默只能担任事故技术代表和现场维护代表，不能代表全体地球居民作政治、文化或永久权利让渡。

## 时间与回滚

- 地球时间与舰上标准时间速率不同，正文首次出现时用简单换算说明，不频繁炫技。
- 主线约 42 个地球日，对应约 17 个舰上标准日。
- 回滚只能回到受信任快照；地球长期数据抽取已污染多数快照。
- 回滚会删除快照后居民的连续体验和事故证词，因此不是伦理中性的备份恢复。
- 不进行全书级时间旅行，不用回滚推翻已发生剧情。

## 不可突破的设定红线

- 不能通过一句命令解决世界危机。
- 不能用“都是仿真”否定人物死亡、伦理或感情后果。
- 不能让专业术语本身成为答案，必须有证据和验证。
- 不能让监管局因感动破例；最终结论必须在规则内成立。
- 不能让赛珞承担所有系统性责任，设计、管理和监管缺口必须分别处理。
