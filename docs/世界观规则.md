# 世界观规则

## 核心设定

1. 地球是高等文明运行的社会仿真项目，但地球内部的人类体验和因果对自身而言真实有效。
2. 歼星舰试射击穿的是地球仿真的底层参数层，而不仅是物理意义上的星球外壳。
3. 现实参数修改权限必须有限、可审计、有代价，不能成为无限许愿能力。
4. 银河监管局可对损坏样本进行保留、隔离、回滚或删除，决定依赖事故等级、修复价值和合规证据。
5. 调查报告不是形式道具：定义、证据、责任边界和整改方案会实际影响监管结论。

## 参数修改暂定规则

- 陈默只能修改已获得权限且能理解其工程含义的参数。
- 修改存在作用范围、持续时间和资源额度。
- 参数相互耦合；局部修复可能产生新的结构、流体、热或社会层异常。
- 系统保留审计日志，但日志可能因事故损坏、权限覆盖或人为篡改而不完整。
- 重大修改需要审批；紧急状态可先执行后补报告，但会增加个人责任。

## 专业与现实映射

- 结构异常：刚度、强度、接触、约束、网格和几何连续性。
- 流体热异常：边界条件、守恒、湍流、传热和相变。
- 多物理场异常：单项修复正确，但耦合后出现意外结果。
- 社会仿真异常：不能简单等同于“人类没有自由意志”；应保留伦理张力。

## 待正式创作前锁定

- 地球项目的创建目的与真实委托方。
- 参数权限的明确额度、界面和副作用。
- 时间回滚是否存在，以及为何不能轻易使用。
- 银河监管局的层级、审查周期和证据标准。
- 歼星舰试射目标、授权链和事故发生机制。
