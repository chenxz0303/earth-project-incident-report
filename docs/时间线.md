# 时间线

> 测试章不进入正式时间线。正式正文以事故发生时刻为 T0，同时记录地球相对日和舰上标准日；具体公历日期不锁定，避免无意义的现实年份限制。

## 时间换算

- 主线跨度：约 42 个地球日。
- 舰上跨度：约 17 个标准日。
- 舰上与地球时间并非固定整数倍，正文只在跨场景时给近似换算。
- 第 126—150 章为连续 72 小时，时间压力必须体现在休息、权限恢复和报告截止中。
