# 章节规划

> 本文件是正式创作前的章节卡，不是小说正文。标题均为暂定，可在写作时调整，但核心事件、证据功能和卷内顺序原则上不随意改动。

## 使用规则

- 每章只承担一个主要剧情任务，最多附带一个人物任务。
- “专业点”必须用普通读者能理解的类比呈现，不写成教材。
- “章尾钩子”用于推动下一章，不等于每章都强行反转。
- 正式写作后，在 `docs/章节摘要.md` 记录实际发生内容；若偏离本规划，必须同步修改相关章节卡、时间线和伏笔表。
