# 人物档案

> 人物信息以支撑长篇连续性为目的。年龄与外貌只保留剧情需要的部分，避免用标签代替人物行为。

## 陈默

- 年龄：29 岁。
- 身份：地球某工程技术公司的仿真工程师；事故后成为地球样本内临时技术顾问和事故技术代表，结局获得受监督的本地项目管理员权限。他不拥有代表全体地球居民让渡权利的政治授权。
- 专业：结构力学、有限元、流体与热仿真；能处理接触、材料、边界条件、网格与流固热耦合，也熟悉验证报告和问题闭环。
- 性格：老实、谨慎、不擅长当面拒绝，外在温吞，内心吐槽密集；面对没有依据的结论会本能不舒服。
- 核心能力：能从条件、参数、结果和报告措辞之间发现不一致；愿意承认不知道，也愿意为可证明的结论负责。
- 弱点：容易默认接下无边界任务；在权力人物面前会紧张；为了避免冲突可能延迟表达反对意见。
- 成长线：被动救火 → 要求明确边界 → 主动叫停高风险方案 → 组织跨专业调查 → 为最终结论签字。
- 权限边界：每日基础额度 12 个因果预算点；只能修改理解且获授权的工程参数；重大修改需要双人复核和审计。
- 禁止写法：无能工具人、特殊血统天选者、全知天才、冷酷战神、随意许愿改写现实。

## 芙娅

- 身份：银河项目体系的事故联络官，负责事故舰、地球项目和监管局之间的程序协调。
- 专业优势：项目法、权限制度、跨文明沟通、证据移交和利益冲突申报。
- 性格：专业克制、重程序、判断快；不擅长理解地球式含蓄表达，偶尔把人情问题处理成表格字段。
- 核心矛盾：她相信程序能保护所有人，却逐步发现程序也可能被利益方设计成封闭回路。
- 成长线：制度执行者 → 主动识别制度漏洞 → 在职责冲突中保持透明 → 以证人身份承担职业风险。
- 与陈默：最初把他视为高风险样本内人员；中期形成专业互补和私人信任；结尾申请成为地球长期联络官。
- 禁止写法：万能翻译器、无条件站队主角、负责所有救场、以恋爱脑放弃职业判断。

## 赵启明

- 身份：陈默在地球公司的项目负责人，约四十岁。
- 性格：会催进度、会说“客户只要结论”，但知道项目出事时不能真把工程师推出去顶锅。
- 能力：擅长客户沟通、资源协调和地球职场流程，技术深度不如陈默但能听懂风险。
- 剧情功能：保留陈默与普通生活的联系；展示并非所有管理者都是脸谱化反派；用地球流程拖住宇宙不合理要求。
- 成长线：把陈默当可靠救火队员 → 意识到长期无边界压任务的伤害 → 作为地球侧见证人承担签字责任。

## 唐可

- 身份：陈默的地球同事，流体与热方向工程师，约二十七岁。
- 性格：说话直接、行动快，面对离谱现象时先做数据清洗再怀疑人生。
- 专业：CFD、气象数据、热网和信号处理；不是陈默的下属，也不承担搞笑跟班功能。
- 剧情功能：补足地球侧流体热主线；证明地球居民可以在不知道全部真相时参与自救。
- 知情边界：第三卷成为受限知情协作者，只知道存在高层技术事故，不完整获知仿真宇宙结构。

## 坎提斯

- 身份：衡光号舰体结构系统工程师。
- 性格：直硬、自尊强、相信量化裕度；早期用“强度未超限”否认结构责任。
- 专业：舰体结构、瞬态动力学、载荷路径和断裂。
- 成长线：保护专业边界 → 承认功能失效不等于强度失效 → 与乌尔澄联合签署耦合结论。
- 与陈默：从质疑地球工程师资格，到尊重陈默对条件和误差边界的敏感。

## 乌尔澄

- 身份：衡光号热流体系统工程师。
- 性格：表面散漫，遇到原始数据问题异常较真；习惯用平均值快速汇报。
- 专业：冷却回路、瞬态流动、传热、采样与控制阀响应。
- 成长线：依赖合格平均值 → 恢复局部高频数据 → 主动承认冷却系统对事故的实际贡献。
- 与坎提斯：长期专业互相嫌弃，最终建立真正的联合模型。

## 赫林

- 身份：衡光号试射负责人。
- 性格：强势、结果导向，能承担压力也习惯把风险推到“会后闭环”。
- 责任：接受拆分验证、压缩系统级复核，对事故负管理责任；没有实施坐标篡改。
- 剧情原则：不能洗白，也不能成为方便的唯一替罪羊。
- 成长线：维护试射团队 → 承认赶工事实 → 公开签署自己的有限责任并提供关键证据。

## 闻铎

- 身份：银河监管局高级事故调查官。
- 性格：冷静、严厉、不接受情绪替代证据；愿意纠正程序错误，也会冻结主角权限。
- 价值观：事实可复核高于任何项目的存亡倾向。
- 剧情功能：代表可信但不温情的监管制度；迫使陈默把直觉转化为证据。
- 成长线：把地球视为受损资产 → 承认其当事方资格 → 支持受监督保留方案。
- 禁止写法：隐藏反派、突然心软的救世主、被陈默几句话说服。

## 洛岑

- 身份：地球项目历史管理员，参与过旧档案迁移。
- 性格：谨慎、多疑、长期独自维护技术债，对自己曾妥协保留旧别名心怀愧疚。
- 经历：发现非法数据抽取后试图举报，被赛珞限制权限；事故前藏入隔离档案区并留下分层死信。
- 剧情功能：连接地球旧别名、非法数据交易和目标覆盖；其证词必须由物证验证，不能一开口解决全部悬疑。

## 赛珞

- 身份：归零服务联合体合规主管，负责项目处置、数据回收和外部审计接口。
- 表面形象：措辞无懈可击、熟悉规则、很少直接撒谎，擅长只回答对自己有利的部分。
- 动机：掩盖长期非法出售地球文明决策数据的业务，避免监管追查牵出客户与内部关系。
- 行为：重新激活地球旧别名、获取一次性批准码、利用四毫秒维护窗口覆盖目标并修改日志；事故后推动删除地球。
- 出场策略：前四卷主要通过合同、插件和程序阻力存在，第 81 章正式露面，第五卷才被证据锁定。
- 禁止写法：无理由毁灭世界、情绪化自曝、能力全靠主角降智。
