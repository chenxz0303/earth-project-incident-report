# 人物档案

## 陈默

- 年龄：约二十九岁。
- 教育与职业：工科硕士，普通仿真工程师。
- 专业：结构力学、有限元、流体与热仿真；接触多物理场耦合和技术报告。
- 性格：有些软弱、闷骚、老实，不擅长当面拒绝，内心吐槽很多。
- 能力边界：专业基础扎实，擅长发现条件、参数和结果之间的不一致；不擅长权力斗争，但能通过证据和报告结构保护自己。
- 成长方向：从被动接任务的工程师，成长为敢于定义问题、承担有限责任的项目管理员。
- 禁止写法：无能工具人、毫无铺垫的全知天才、冷酷战神、靠权限随意改写现实。

## 芙娅

- 身份：外星事故联络官。
- 职责：协调地球、事故舰和银河监管体系之间的信息、权限和流程。
- 性格方向：专业、克制、重程序，但并非机械冷漠；对地球职场逻辑既困惑又逐渐理解。
- 与陈默关系：先是责任绑定与互相观察，后发展为慢热、点到为止的轻微暧昧。
- 功能边界：她不是万能翻译器或救场机器；她掌握程序与组织信息，陈默掌握工程判断和地球经验，两人能力互补。

## 待设计角色槽位

- 陈默的地球项目负责人：承担典型职场压力与现实牵挂，不脸谱化。
- 歼星舰试射负责人：事故责任链核心人物之一，动机待证据链确定。
- 结构系统工程师：与陈默在有限元和失效模式上形成专业碰撞。
- 热流体系统工程师：负责另一条主要调查线。
- 银河监管调查官：代表审查规则与删除风险，不能只是反派。
- 地球项目历史管理员：连接旧项目记录和长期伏笔。
