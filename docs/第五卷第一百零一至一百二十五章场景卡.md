# 第五卷第一百零一至一百二十五章场景卡

> 本文件是第五卷详细场景大纲，不是小说正文。本卷主题是：事故调查不能用“所有人都有责任”含糊收尾，必须证明每一层错误能造成什么、不能造成什么，以及谁完成了最后的人为覆盖。

## 卷内结构

- 101—108：成立联合任务组，用失效树拆开结构、热流体、旧别名和人为权限。
- 109—117：追查批准码与四毫秒维护窗口，恢复洛岑证词和非法买方链。
- 118—125：赛珞反向指控，完整复现事故，阻止灭证并进入地球根裂纹倒计时。

## 第五卷责任链结论

| 层级 | 已证明行为 | 责任边界 |
|---|---|---|
| 设计层 | 隐藏热漂移补偿项、批准码缓存、接口披露不足 | 能造成偏航和安全防线削弱，不能单独造成精准命中 |
| 管理层 | 赫林批准拆分验证、把接口风险留到试射后闭环 | 让设计缺陷未经系统级验证进入试射，不等于实施篡改 |
| 篡改层 | 赛珞激活旧别名、借取批准码、利用维护窗口覆盖目标并灭证 | 直接决定地球被精准命中，是主要故意责任 |
| 监管层 | 采购与关联机构未识别数据交易利益冲突 | 形成长期监督漏洞，不替代具体行为人的责任 |

## 第五卷结束状态

- 四层事故责任链已形成可重复证据，赛珞受到控制但案件尚未最终裁决。
- 芙娅暂停联络职务，以证人身份准备最终听证。
- 陈默成为实际调查负责人，同时仍受越权审查和权限额度约束。
- 地球根裂纹进入不稳定扩展，只剩72小时完成修复与最终报告。
- 第六卷必须并行完成现实修复、监管听证和事故报告定稿。
