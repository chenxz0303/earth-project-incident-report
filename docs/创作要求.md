# 创作要求

## 类型与基调

类型为科幻、职场喜剧、悬疑调查、轻松爽文。整体偏沙雕爽文，笑点较密集；关键悬疑、危险和情感段落必须认真，不用玩梗破坏可信度。

笑点主要来自职场甩锅、审批流程、报告修改、宇宙规则与地球职场之间的反差，避免单纯堆砌网络梗。

## 叙事要求

- 使用第三人称，主要跟随陈默。
- 陈默的内心吐槽可以密集，但外在反应应符合他老实、谨慎、不擅长当面拒绝的性格。
- 爽点来自专业判断、有限权限下的聪明解决方案和报告博弈，不靠突然开挂碾压。
- 悬疑线索必须可回溯，重要结论要有前置证据。
- 感情线慢热、点到为止，不喧宾夺主。

## 主角边界

陈默约二十九岁，工科硕士，普通仿真工程师，熟悉结构力学、有限元、流体与热仿真，也接触多物理场耦合和技术报告。他有些软弱、闷骚、老实，但绝不无能；不得突然变成冷酷战神，也不能为了制造笑点让他丢失基本职业能力。

## 专业内容原则

结构力学和流体热仿真并重。可使用有限元、接触、材料参数、边界条件、网格、不收敛、流固耦合等概念，但每次出现都要满足至少一项：推动剧情、制造合理笑点、建立线索、解决危险。

专业解释优先使用日常类比，再给必要术语；不写成教材，不用大段参数炫技。

## 规模

- 预计 5—7 卷。
- 共 140—170 章。
- 每章约 3000—4000 字。
- 总字数约 50 万字。

## 每章完成清单

- 更新章节摘要。
- 更新时间线。
- 更新人物状态。
- 记录新增伏笔及回收伏笔。
- 更新关键设定变化。
- 检查专业概念是否让普通读者能够理解。
- 运行项目一致性测试。
