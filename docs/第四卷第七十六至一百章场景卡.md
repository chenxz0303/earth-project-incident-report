# 第四卷第七十六至一百章场景卡

> 本文件是第四卷详细场景大纲，不是小说正文。本卷主题是：当修改权限被冻结、证据来源受质疑时，陈默必须先证明地球有资格参与定义自己的事故。

## 卷内结构

- 076—083：闻铎进场、证据重新编号、地球争取观察席。
- 084—091：恢复原始数据、追查洛岑与处置合同、芙娅职责冲突。
- 092—100：恢复死信、突破合法墙、预听证取得临时当事方资格。

## 第四卷结束状态

- 地球获得临时当事方身份，删除程序暂停 14 日。
- 闻铎认可陈默的证据方法，但未承诺保留地球。
- 芙娅因利益冲突受到限制，与陈默的信任需要在职责透明中重建。
- 洛岑确认存活，记忆接口受损；赛珞及归零服务成为明确调查对象。
- 下一卷必须用可复现实验区分设计缺陷、管理过失和人为覆盖的贡献。
