# 第六卷第一百二十六至一百五十章场景卡

> 本文件是第六卷详细场景大纲，不是小说正文。本卷主题是：陈默不能靠隐瞒事故保住地球，他必须证明地球值得保留、能够继续运行，并给出比删除更可控的风险方案。

## 卷内结构

- 126—135：制定72小时方案，完成第一阶段修复与责任方签字。
- 136—145：最终听证和根裂纹修复并行，地球完成自审计能力证明。
- 146—150：监管表决、责任处理、陈默成为受监督管理员并留下续作钩子。

## 三条并行主线

| 主线 | 截止条件 | 核心成果 |
|---|---|---|
| 地球修复 | 根裂纹在72小时后失稳 | 隔离抽取器、闭合裂纹、验证剩余风险 |
| 监管听证 | 最终处置表决前 | 证明地球是受害者、证人、修复参与者和不可替代项目 |
| 事故报告 | 表决前完成签署 | 不撒谎地分配四层责任，并提出可审计保留方案 |

## 最终监管结论必须包含

- 地球是事故受害者、关键证据载体和修复参与者，删除会造成证据与主体双重损失。
- 地球居民产生了创建模型无法预设的新知识，备份无法替代其连续历史。
- 地球已证明分布式自治、纠错和自审计能力，风险可通过隔离、监测和受监督管理员控制。
- 保留不是免除风险，而是比删除更符合比例原则和证据保全要求。
